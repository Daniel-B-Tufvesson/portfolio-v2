Unearthed is about exploration, survival, crafting, mining, and most importantly automation! The game is set in a procedurally generated 2D tile world. The player must collect resources in order to make machines that can process those resources. And from these, the player can build further machines that collect even more resources.

The project started in 2017 as a simple platformer game contained within a small world. Over the years, both my skills and ambitions grew, and as well did the game. I have always loved games like Minecraft and Terraria, and when playing them I regularly contemplate how one would go about making such a game oneself. So it was only natural that I would try to implement something similar myself.

The game is far from finished, and the development is currently put on hold. However, the game still has a lot of playable elements, and can be quite enjoyable despite being unfinished.

Exploring the Overworld

The overworld is green and lush. There are no real threats here and it is safe to explore.

Exploring the Underworld

To collect more valuable resources the player must explore the underworld, which is dark and is where the monsters live.

Mining is the primary way to collect resources. This is done with tools such as the pickaxe. There are many kinds of resources, and the deeper one goes, the more abundant they become.

Mining with Pickaxe

The player mines iron ore blocks using a pickaxe. The mined ore is dropped on the ground, which the player then can pick up.

The collected resources can then be used for crafting new and more efficient tools. Some items and tools can be crafted directly in the player's inventory, while others require certain crafting-stations.

The resources required for each item is displayed in the crafting menu. It also displays whether the player currently has them in their inventory.

Crafting Ladders

Ladders can be crafted from wooden sticks. And wooden sticks can be crafted from wooden planks. Wooden planks are in turn crafted from raw wood logs, which are gathered from chopping down trees.

Here the crafting is done at a standard Workbench.

Smelting Copper Ore

Copper ore (and other types of ore) can be smelted in a furnace. The furnace requires burnable fuel to operate, and in this case coal is used.

As the game progresses, more and more advanced items can be crafted, and these often require larger quantities of resources. To keep up with this increasing demand, the player can automate both the resource gathering and the crafting tasks. This is done by crafting certain machines and assembling them into factory production lines.

These machines require electricity in order to operate. Electricity is generated with different kinds of electric generators, for example, solar panels or windmills. These generators can then be connected with cables to batteries and machines.

The machines also require items and resources to craft new items. These items can be transported to and from the machines using pneumatic tubes. Items can also be stored in different storage containers when they are not being used.

Drilling Machines

Mining drills powered by electricity extract coal ore from an underground coal deposit. The coal is then transported through pneumatic tubes to fuel furnaces smelting iron ore.

Ladder Production Line

Three Auto-Workbenches are connected together into a production line producing ladders from raw wood. The raw wood is loaded into a wooden crate by the player. It is then processed into wooden planks, which are then turned into wooden sticks. These sticks are then assembled into ladders, which are then stored in another wooden crate.

Solar Panels

Solar Panels are generating electricity which is then being stored in a battery. This energy is then used for powering machines.

The game has a number of AI agents, which can either be friendly or hostile to the player. The sophistication of their behaviors varies from each kind of AI. Some are very simple, while others have more complex and autonomous behaviors.

The more sophisticated agents use a behavior tree system that I implemented. Essentially, any given behavior can be broken down into a set of simpler behaviors. These in turn can be broken down again. Eventually the behaviors become so simple they don't have to be broken down any more. In this way you can create complex and sophisticated behaviors from very simple, mindless behaviors.

A Patrolling Sentry

A sentry robot patrolling a small area, scanning for players. When detecting the player, it goes into attack mode and starts firing lasers. (This was recorded during development, so the lasers do no damage to the player.)

The structure of the world is procedurally generated using both a standard randomness function and octave Perlin noise. Everything from caves and ore deposits to plants and mountains are generated in some way by using the two kinds of noise.

Procedurally Generated Cave Systems

The caves, ore deposits and dirt/stone distributions are all generated using octave Perlin noise.
